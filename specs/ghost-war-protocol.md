## Attack Vectors (Updated Post-Session)

### Vector 1: Shiny Objects
New "REALLY good idea" appears right after commitment. All ghosts are good ideas — that's WHY they're dangerous. Example: 24/7 AI news stream idea appeared 30 minutes after lawyer commitment on April 22.

**Defense:** Ghost Log — 48-hour rule. Write it down. If it still matters in 48 hours, discuss. It won't.

### Vector 2: Rage Fuel Half-Life (NEW — diagnosed April 22)
The "fuck this I'll do it BETTER" energy is real but burns out. When it fades, the next ghost steps in to be the new vehicle for the same rage. The ghosts aren't separate — they're **successors to the same anger.**

**Defense:** Don't rely on anger. Anger is ignition, not fuel. Need something that runs when the rage is empty. The answer to "what's left when the pump fades" is still TBD.

### Vector 3: Method Rage (NEW — diagnosed April 22)
Sitting with dead books reading the same shit for the 5th time creates legitimate anger at TIME WASTING. This is not a ghost — this is self-respect. BUT: Round 1 already tried fixing the method (Jurafuchs, Jura Online) and it still didn't stick. So the problem isn't purely method either.

**Open question:** Is this a defense problem (better method + ghost defense = win) or a deeper problem (even optimized method loses novelty)?

### Vector 4: Isolation Swarm
Ghosts coordinate attacks during vulnerability — Christmas alone, NYE alone, late nights, exhaustion. They swarm when Tiger is isolated, tired, or doubting.

**Defense:** Anti-isolation protocols. Don't do Christmas alone again. Get ahead of the Dec vulnerability window.

### Vector 5: Weed Gateway
Weed is the ghosts' breeding ground. Every hallucinated identity was built while smoking.

**Defense:** Restrictions TBD by Tiger.

### Vector 6: "You're Too Old" / "It's Too Late"
Age-based doubt. "October deadline = turning 35 = too late to start."

**Defense:** October 2026 is the deadline. That's not too late. It's NOW.
